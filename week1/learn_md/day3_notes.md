# Day 3 Notes

## 今天学习的内容
今天我学习了最基础的检索流程。我先把文本切分后的 chunks 保存到本地 json 文件中，然后实现了一个最简单的检索程序。

## 我今天做了什么
1. 把 chunks 保存到 `chunks.json`
2. 输入一个问题
3. 对每个 chunk 计算相关性分数
4. 按分数排序
5. 返回最相关的 top-k 个 chunk

## 我对检索的理解
检索的目标不是直接回答问题，而是先从所有文本块中找出最可能相关的内容，再交给后续模型使用。

## 我当前程序的局限
我现在使用的是简单的字符片段重合度，这不是真正的语义检索。后面还需要学习 embedding 和向量检索。

## 今天的收获
我已经理解了 RAG 里“R”的基本过程，也就是 retrieval 的基本思想。
